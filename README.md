# Programação Orientada a Objetos 👩🏽‍💻

### DESCRIÇÃO

Repositório com avaliações, listas de exercícios e projetos da disciplina de Programação Orientada a Objetos, desenvolvidos em Java. Os conteúdos estão organizados conforme os tópicos abordados em sala de aula.

### OBJETIVO

Este repositório tem como objetivo exercitar os conteúdos abordados em sala de aula, contribuindo para o desenvolvimento das minhas habilidades ao longo da disciplina.

### ESTRUTURA

**aula-2-exercicios:** Exercícios de entrada, saída e operações matemáticas, propostos em sala de aula, com foco na prática da sintaxe do Java.

### TECNOLOGIA UTILIZADAS

- Java
- IntelliJ / VS Code
- JDK
- Git e Github

### COMO EXECUTAR

1. Certifique-se de que o JDK está instalado e que as variáveis de ambiente estejam configuradas
2. Abra o terminal e compile o arquivo: javac NomeDoArquivo.java
3. Execute o arquivo: java NomeDoArquivo

### OBSERVAÇÕES

Os exercícios fazem parte do meu aprendizado acadêmico e, como todo universitário, ainda estou aprendendo. Isso significa que os códigos podem ser atualizados conforme meus conhecimentos evoluem. 😊
